Bugs -

11. Calendar Grid vs Detail Panel Show Different Tasks
	- renderCalendar() vs showCalDay(), ~lines 3420 and 3475
	- Grid filters tasks by dueDate first, falling back to createdAt. Detail panel (showCalDay) only filters by createdAt, ignoring dueDate. Task created March 1st with due date March 15th appears in March 15 grid cell but clicking shows nothing.
	- Apply same dueDate-first logic to showCalDay

12. Drag-and-Drop State Leak Between Systems	
	- taskDragStart() vs matrixDragStart(), ~lines 2230 and 3380	
	- Two separate drag variables (taskDragId and matrixDragId) with dropToQuadrant only checking taskDragId. Matrix drops may fail or use wrong ID when both systems active.	
	- Unify drag state to single currentDragId variable; check in all drop handlers

13. Duplicate Subject Names (Case-Sensitive)	
	- addSubject(), ~line 2515	
	- Only checks exact match: "Math" vs "math" allowed due to case sensitivity.	
	- Normalize to lowercase for comparison: name.toLowerCase()

14. Exam Date Validation Bypass	
	- openAddExam() sets min attribute, but submitExam() lacks validation, ~line 2670	
	- HTML min attribute can be bypassed with inspector; no server-side-style validation.	
	- Add in submitExam(): if (new Date(date) < new Date().setHours(0,0,0,0)) { showToast("⚠️", "Invalid date", "Cannot add exam in the past"); return; }

15. Session Counter Never Resets	
	- onTimerComplete() pomodoro logic, ~line 2790	
	- sessionCount increments indefinitely; after 4 sessions → long break → 4 more → long break, but counter keeps growing. No modulo reset means sessionCount % n works mathematically but value grows unbounded.	
	- Add sessionCount = 0 after long break, or use sessionCount++ % sessionsBeforeLong pattern

16. Session Title Not Persisted During Session	
	- session-title-input, ~line 2725	
	- User enters session title, but if page refreshed during active session, title is lost (not in data object).	
	- Store currentSessionTitle in data object; restore on init

17. Chart Re-creation on Every Render	
	- renderAnalytics() and renderGrades(), ~lines 2930 and 3130	
	- Charts destroyed and recreated on every navigation — expensive operation, visual flicker.	
	- Use chart.update() instead of destroy/create when data changes

18. Quote Index Not Restored	
	- Increase the quote count from 100 to 250
	- data.quoteIndex, ~line 1870	
	- Saved to localStorage but init() doesn't restore it; always starts at 0.	
	- Add `data.quoteIndex = data.quoteIndex

19.  AudioContext Created Per Sound	
	- playSound(), ~line 3540	
	- New AudioContext each call violates best practices; causes resource exhaustion.	
	- Reuse single AudioContext instance (already fixed in code with _audioCtx but verify)

20. Missing Notification Permission	
	- Not implemented	
	- Modern apps should request notification permission for deadline alerts.	
	- Add Notification.requestPermission() on user action; schedule reminders
